# WhatsApp CRM

## Setup

1. Clone the repo
2. cd into `packages/backend` and run `npm install`
3. Copy `.env.example` to `.env` and fill in your Meta credentials
4. Run `node index.js` to start the backend
5. cd into `frontend` and run `npm install` then `npm start`
6. Expose backend with `ngrok http 3001` and update Meta webhook URL
